# Retail-Customer-Retention-Analysis
Project Overview
This project focuses on analyzing retail customer data to identify key drivers of customer churn and develop strategies for retention. By integrating demographic, transactional, and loyalty data, I have transformed raw datasets into an interactive dashboard that provides actionable business insights.
End-to-End Data Pipeline: Performed data cleaning and Exploratory Data Analysis (EDA) to address null values and inconsistent formats.
Interactive Visualization: Developed a comprehensive Power BI dashboard to track customer churn and loyalty trends.
Actionable Insights: Identified high-risk customer segments and delivered visual reports to optimize retention strategies.
